# Shed Boat

Project for the autonomous boat controller.

## Hardware

We have a uBlox M8 GPS and Compass module. The GPS runs at 9600Bd

| Pin colour   | Function |
|:-------------|:---------|
| Red          | `+5v`    |
| Black        | `Gnd`    |
| Green        | `SCL`    |
| Blue         | `SDA`    |
| Yellow       | `rx`     |
| White        | `tx`     |
